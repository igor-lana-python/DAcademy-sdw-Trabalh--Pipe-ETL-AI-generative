## 🧠 Sobre o Projeto

Este projeto implementa um pipeline **ETL (Extract, Transform, Load)** integrado com **IA Generativa**, desenvolvido durante atividades da **DIO (Digital Innovation One)**.

A proposta é unir **engenharia de dados** com **inteligência artificial**, automatizando a transformação e o enriquecimento de dados de forma inteligente.

---

## ⚙️ Arquitetura do Projeto

```
Extract → Transform → Load
        ↓
   IA Generativa
```

---

## 🔍 Etapas do Pipeline

### 📥 Extract

* Leitura de dados (CSV, JSON, etc.)
* Coleta de informações de diferentes fontes

### 🔄 Transform

* Limpeza e tratamento dos dados
* Padronização
* Enriquecimento com IA generativa:

  * Geração de descrições
  * Criação de insights automáticos

### 📤 Load

* Exportação dos dados processados
* Prontos para análise ou uso em outros sistemas
